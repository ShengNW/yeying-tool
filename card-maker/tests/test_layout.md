# 手工测试清单

## 基本输出
- [ ] `card-maker --preset month --title 月卡 --out output/a.png` 可生成图，分辨率为 1920×1080，16:9，≤5MB。

## 尺寸与比例
- [ ] 指定 `--width 1280` 或 `--height 720`，另一边自动匹配 16:9，且均 ≥480。

## 文本
- [ ] 标题短字（2–4字）和长字（10–16字）都能自适应字号，最多两行。
- [ ] bullets 1–6 条，长句自动换行且缩进对齐。

## 视觉
- [ ] 背景深色/浅色下，前景字对比度足够；下划线与小方块装饰正常。
- [ ] 信息卡阴影柔和，圆角正确，水印（如设置）在右下角。

## 体积
- [ ] PNG/JPG 导出均 ≤5MB。JPG 会自动降低质量，PNG 会优化并在不足时等比缩小。

## 预设
- [ ] `examples/*.json` 均可通过 `--params-json` 生成对应图片。
